Getting started

This guide walks you through the core Hopper workflow: opening a single archival video, drawing a region around the burnt-in cue display, and running your first analysis to build the cue map.

Open a video

Launch Hopper from your Applications folder. From the welcome screen, click New Project and use the file picker to select your archival video file. Hopper supports the video formats macOS can play natively — MP4, MOV, and MXF files all work well.

Once the video is loaded you will see it appear in the playback window. Take a moment to scrub through the recording and confirm the cue number display is visible in the frame — you will need to be able to see it clearly to draw an accurate detection region.

Draw a cue region

Hopper reads cue numbers directly from the video using on-device OCR. To tell it where to look, you need to draw a rectangle over the part of the frame where the cue number is displayed.

  1. Pause playback on a frame where a cue number is clearly visible.
  2. Click the Draw Region button (or select it from the menu). The cursor changes to a crosshair.
  3. Click and drag to draw a box tightly around the cue number display. The box should contain the number but as little surrounding content as possible — a tight crop gives the OCR engine the best chance of reading it accurately.
  4. Release the mouse to confirm the selection.

The region is stored as fractional coordinates relative to the video frame, so it will work correctly even if you open the same video at a different resolution later.

Run analysis

With a region drawn, click Analyse Video. Hopper scans the recording frame by frame within the selected region, detects every cue number transition, and records the exact timestamp each cue first appeared.

Analysis runs entirely on your Mac — no internet connection is needed. Progress is shown in a HUD overlay; on a typical show recording (60–90 minutes) this usually completes in a few minutes.

When complete, the results are saved in a small .hopper sidecar file placed alongside your video. On subsequent opens, Hopper loads the sidecar automatically so you will not need to re-analyse unless the video or cue region changes.

Next steps

Once analysis is complete, the cue list will populate in the sidebar. From here you can:

  • Click any cue to jump the video to that moment.
  • Connect an ETC EOS console and enable Console Follow so Hopper seeks automatically when a cue fires on the board — see Triggering the console.
  • Configure OSC settings and remote control in Settings.
